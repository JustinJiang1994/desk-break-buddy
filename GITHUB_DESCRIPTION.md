## 项目简介（用于GitHub README顶部）

```markdown
# 🏃 desk-break-buddy

> 桌面休息伙伴 - 一款帮助您养成健康办公习惯的Chrome浏览器插件

[![Chrome Web Store](https://img.shields.io/badge/Chrome-Extension-green)](https://chrome.google.com/webstore)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Version](https://img.shields.io/badge/version-1.0.0-orange)](https://github.com/yourusername/desk-break-buddy)

通过定时提醒功能，帮助长时间使用电脑的用户规律起身活动和补充水分，缓解久坐疲劳，养成健康的办公习惯。

## ✨ 主要特性

- 🎯 **双提醒类型** - 起身活动提醒 + 喝水提醒
- ⏰ **自定义计时器** - 创建任意数量的个性化提醒
- ☁️ **云端同步** - 数据自动同步到所有设备
- 🔔 **智能提醒** - 弹窗 + 声音双重提醒
- 📊 **使用记录** - 查看近7天的提醒历史
- ⏸️ **暂停功能** - 灵活控制提醒时间
- 🎨 **简洁界面** - 清新简约的设计风格

## 🚀 快速开始

### 安装方法

1. 下载或克隆本项目
2. 打开Chrome浏览器，访问 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹

详细安装说明请查看 [INSTALL.md](INSTALL.md)

### 使用说明

1. 点击浏览器工具栏的插件图标
2. 在快速操作面板中开启提醒功能
3. 进入设置页面自定义提醒间隔
4. 创建自定义计时器满足个性化需求
```
