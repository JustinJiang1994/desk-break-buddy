# Chrome应用商店打包说明

## 打包文件

已创建 `desk-break-buddy.zip` 文件，包含上传到Chrome应用商店所需的所有文件。

## 包含的文件

### 核心文件
- `manifest.json` - 插件配置文件
- `background.js` - 后台服务脚本

### 界面文件
- `popup.html/js/css` - 快速操作面板
- `options.html/js/css` - 设置页面
- `notification.html/js/css` - 提醒弹窗

### 资源文件
- `icons/*.png` - 所有图标文件（16x16, 48x48, 128x128，包括正常、激活、暂停状态）
- `sounds/` - 音效文件目录（需要用户自行添加音效文件）

## 上传到Chrome应用商店

### 步骤

1. **访问Chrome Web Store开发者控制台**
   - 访问：https://chrome.google.com/webstore/devconsole
   - 使用您的Google账户登录

2. **创建新项目**
   - 点击"新增项目"
   - 填写项目信息

3. **上传ZIP文件**
   - 点击"上传ZIP文件"
   - 选择 `desk-break-buddy.zip`
   - 等待上传和验证完成

4. **填写商店信息**
   - **名称**：桌面休息伙伴
   - **简短描述**：定时提醒您起身活动和补充水分，养成健康的办公习惯
   - **详细描述**：参考README.md中的内容
   - **图标**：使用 `icons/icon128.png`
   - **截图**：可以添加使用截图（可选）

5. **设置分类和隐私**
   - 选择适当的分类
   - 填写隐私政策（如果需要）
   - 设置可见性（公开/未列出）

6. **提交审核**
   - 检查所有信息
   - 点击"提交审核"
   - 等待Google审核（通常需要几天）

## 注意事项

⚠️ **重要提示**：
- ZIP文件大小限制：通常不超过10MB
- 当前ZIP文件大小：约67KB（未压缩前约107KB）
- 如果添加音效文件，注意文件大小
- 确保所有图标文件都已包含
- 音效文件是可选的，如果缺失不会影响基本功能

## 文件大小优化

如果ZIP文件过大，可以考虑：
- 压缩图标文件（使用更小的PNG）
- 压缩音效文件（使用更小的MP3）
- 移除不必要的文件

## 更新版本

更新插件时：
1. 修改 `manifest.json` 中的版本号
2. 重新打包ZIP文件
3. 在开发者控制台上传新版本
