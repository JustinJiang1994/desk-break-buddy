# 权限使用说明（中文版 - 用于Chrome应用商店审核）

## 权限申请及使用理由

### 1. storage（存储权限）

**申请理由：**
本插件需要存储用户的个性化设置和提醒记录，以实现以下功能：

1. **保存用户设置**
   - 保存起身活动提醒的时间间隔设置
   - 保存喝水提醒的时间间隔设置
   - 保存用户创建的自定义计时器配置（名称、间隔、启用状态）
   - 保存声音提醒的开关状态和音效类型选择

2. **数据云端同步**
   - 使用 `chrome.storage.sync` API 将用户设置同步到Chrome账户
   - 让用户在不同设备间自动同步设置，无需重复配置
   - 数据存储在用户的Chrome账户中，完全私有和安全

3. **提醒历史记录**
   - 记录近30天的提醒触发情况
   - 帮助用户了解使用习惯，查看提醒统计

**数据安全：**
- 仅存储插件功能相关的设置数据
- 不收集任何用户个人信息或浏览数据
- 数据完全存储在用户的Chrome账户中，不会传输给第三方

### 2. alarms（定时器权限）

**申请理由：**
这是实现定时提醒功能的核心权限，用于：

1. **创建定时提醒**
   - 使用 `chrome.alarms` API 创建后台定时器
   - 支持多个独立的定时器同时运行：
     * 起身活动提醒定时器（默认60分钟）
     * 喝水提醒定时器（默认30分钟）
     * 用户自定义的多个计时器（可设置任意时间间隔）

2. **后台运行**
   - 定时器在浏览器后台持续运行
   - 即使用户关闭了所有标签页，提醒功能仍然有效
   - 不占用用户当前浏览页面的资源，不影响浏览体验

3. **精确触发**
   - 确保提醒在设定的时间准确触发
   - 支持用户自定义任意时间间隔（至少1分钟）
   - 自动计算下次提醒时间，循环提醒

**使用场景示例：**
- 每60分钟提醒用户起身活动
- 每30分钟提醒用户补充水分
- 用户自定义的提醒（如每2小时的眼保健操、每3小时的休息等）

### 3. notifications（通知权限）

**申请理由：**
用于在提醒时间到达时，通过弹窗方式提醒用户：

1. **显示提醒窗口**
   - 创建独立的提醒弹窗，确保用户能够注意到提醒
   - 弹窗居中显示，设计简洁美观
   - 包含清晰的文字提示和图标

2. **不同类型的提醒**
   - **起身活动提醒**：弹窗强制显示30秒，无法手动关闭，确保用户短暂中断久坐状态
   - **喝水提醒**：弹窗可手动关闭，提供"跳过"按钮
   - **自定义提醒**：弹窗可手动关闭，显示自定义计时器名称

3. **用户体验优化**
   - 提醒窗口不干扰用户正常浏览
   - 支持声音提醒（可选）
   - 提醒内容清晰明确，帮助用户养成健康习惯

## 权限使用范围

本插件**仅申请并使用**上述三个权限，且**仅用于**实现定时提醒功能：

- ✅ **storage** - 仅用于保存和同步用户设置，不访问任何网站数据
- ✅ **alarms** - 仅用于创建和管理定时提醒，不执行任何其他操作
- ✅ **notifications** - 仅用于显示提醒弹窗，不发送任何其他通知

## 隐私保护承诺

- 🔒 **数据安全**：所有数据存储在用户的本地设备或Chrome账户中
- 🔒 **隐私保护**：不收集、不传输任何用户个人信息
- 🔒 **无追踪**：不访问任何网站内容，不跟踪用户浏览行为
- 🔒 **离线工作**：核心功能完全离线，无需网络连接（数据同步除外）
- 🔒 **透明使用**：所有权限使用完全透明，仅用于声明功能

## 功能说明

**桌面休息伙伴**是一个专注于健康提醒的Chrome扩展插件，旨在帮助长时间使用电脑的用户：

- 🏃 规律起身活动，缓解久坐疲劳
- 💧 定时补充水分，保持身体健康  
- ⏰ 创建个性化提醒，满足不同需求

所有功能都围绕"健康提醒"这一核心目标，权限使用完全必要且透明。
