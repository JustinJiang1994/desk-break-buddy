# Chrome应用商店权限说明

## 权限使用说明

### 1. storage（存储权限）

**使用理由：**
- 保存用户的提醒设置（起身活动提醒间隔、喝水提醒间隔）
- 保存用户的自定义计时器配置（名称、间隔时间、启用状态）
- 保存声音设置（是否启用、音效类型选择）
- 保存提醒历史记录（近30天，用于统计和展示）
- 保存暂停状态信息
- 使用 `chrome.storage.sync` API 实现数据云端同步，让用户在不同设备间自动同步设置

**数据存储内容：**
- 仅存储插件功能相关的设置数据
- 不收集任何用户个人信息
- 不访问任何网站数据
- 数据存储在用户的Chrome账户中，完全私有

### 2. alarms（定时器权限）

**使用理由：**
- 实现定时提醒功能的核心权限
- 使用 `chrome.alarms` API 创建和管理定时器
- 支持多个独立的定时器：
  - 起身活动提醒定时器
  - 喝水提醒定时器
  - 用户自定义的多个计时器
- 确保提醒在后台准确触发，即使用户关闭了浏览器标签页
- 定时器在浏览器后台运行，不占用用户当前浏览页面的资源

**使用场景：**
- 每60分钟触发一次起身活动提醒
- 每30分钟触发一次喝水提醒
- 用户自定义的任意时间间隔提醒（如每2小时的眼保健操提醒）

### 3. notifications（通知权限）

**使用理由：**
- 在提醒时间到达时，通过弹窗提醒用户
- 创建独立的提醒窗口，确保用户能够注意到提醒
- 提醒窗口居中显示，包含清晰的文字提示
- 支持两种提醒方式：
  - 起身活动提醒：强制显示30秒，确保用户短暂中断久坐
  - 喝水提醒和自定义提醒：可手动关闭

**用户体验：**
- 提醒窗口设计简洁美观，不干扰用户正常浏览
- 提醒内容清晰明确，帮助用户养成健康习惯
- 支持跳过功能，给用户灵活的选择

## 权限使用范围

本插件**仅使用**上述三个权限，且**仅用于**实现定时提醒功能：

- ✅ **storage** - 仅用于保存和同步用户设置
- ✅ **alarms** - 仅用于创建和管理定时提醒
- ✅ **notifications** - 仅用于显示提醒弹窗

## 隐私保护

- 🔒 所有数据存储在用户的本地设备或Chrome账户中
- 🔒 不收集、不传输任何用户个人信息
- 🔒 不访问任何网站内容或数据
- 🔒 不跟踪用户浏览行为
- 🔒 完全离线工作，无需网络连接（除数据同步外）

## 功能说明

本插件是一个**健康提醒工具**，旨在帮助用户：
- 规律起身活动，缓解久坐疲劳
- 定时补充水分，保持身体健康
- 创建个性化提醒，满足不同需求

所有功能都围绕"健康提醒"这一核心目标，权限使用完全透明且必要。
