# 图标文件说明

请在此目录下放置以下图标文件：

- `icon16.png` - 16x16 像素图标
- `icon48.png` - 48x48 像素图标  
- `icon128.png` - 128x128 像素图标
- `icon16-active.png` - 16x16 像素激活状态图标（提醒时显示）
- `icon48-active.png` - 48x48 像素激活状态图标
- `icon128-active.png` - 128x128 像素激活状态图标
- `icon16-paused.png` - 16x16 像素暂停状态图标
- `icon48-paused.png` - 48x48 像素暂停状态图标
- `icon128-paused.png` - 128x128 像素暂停状态图标

图标设计建议：
- 主图标：包含"休息"相关元素，如杯子、起身图标等
- 激活状态：可以使用不同颜色或闪烁效果
- 暂停状态：可以使用灰色或暂停符号

可以使用在线图标生成工具或设计软件创建这些图标。
