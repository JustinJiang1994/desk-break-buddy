# 快速测试指南

## 手动加载插件步骤

1. **打开Chrome扩展程序页面**
   - 访问：`chrome://extensions/`
   - 或：菜单 → 更多工具 → 扩展程序

2. **开启开发者模式**
   - 点击右上角的"开发者模式"开关

3. **加载插件**
   - 点击"載入未封裝項目"（加载未打包项目）按钮
   - 在文件选择对话框中选择项目文件夹：`/Users/justin/code/desk-break-buddy`
   - 点击"选择文件夹"

4. **处理图标错误（可选）**
   - 如果看到图标文件缺失的错误，这是正常的
   - 功能仍然可以正常使用
   - 图标会显示为默认图标

## 快速功能测试

### 测试1：快速操作面板
1. 点击浏览器工具栏的插件图标
2. 应该看到：
   - "桌面休息伙伴"标题
   - 起身提醒和喝水提醒的状态显示
   - 两个开关按钮
   - "暂停提醒"和"进入设置"按钮

### 测试2：设置页面
1. 点击"进入设置"按钮
2. 应该看到设置页面，包含：
   - 提醒时间设置
   - 声音设置
   - **自定义计时器**区域
   - 提醒记录

### 测试3：创建自定义计时器
1. 在设置页面，点击"+ 添加计时器"
2. 输入：
   - 名称：测试提醒
   - 间隔：1（分钟，用于快速测试）
3. 点击"保存"
4. 应该看到新计时器出现在列表中
5. 测试开关、编辑、删除功能

### 测试4：提醒功能（快速测试）
1. 在设置页面：
   - 将起身提醒间隔改为1分钟
   - 将喝水提醒间隔改为1分钟
   - 保存设置
2. 等待1分钟
3. 应该看到提醒弹窗出现
4. 测试跳过按钮和自动关闭功能

### 测试5：暂停功能
1. 在快速操作面板点击"暂停提醒"
2. 选择"15分钟"
3. 检查状态是否显示"暂停中"
4. 等待15分钟后检查是否恢复

## 调试方法

如果遇到问题，可以：

1. **查看后台脚本日志**：
   - 在扩展程序页面找到插件
   - 点击"service worker"链接
   - 查看控制台

2. **查看弹窗日志**：
   - 右键点击插件图标
   - 选择"检查弹出内容"

3. **查看设置页面日志**：
   - 在设置页面右键
   - 选择"检查"
